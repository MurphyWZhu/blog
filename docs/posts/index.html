在Archlinux上安装Podman

Podman podman是一个无守护进程的容器引擎，它还可以实现无根运行容器，现在最新版本的红帽系Linux都默认使用podman。 在Archlinux上安装podman 安装podman sudo pacman -S podman crun 配置无根运行 sudo touch /etc/subuid /etc/subgid sudo usermod --add-subuids 100000-165536 --add-subgids 100000-165536 username #将username替换为用户名 查看podman信息 podman info 配置podman镜像仓库 新建registries.conf配置文件 mkdir -p $HOME/.config/containers vim $HOME/.config/containers/registries.conf 将以下内容写入registries.conf unqualified-search-registries = ["docker.io"] [[registry]] prefix = "docker.io" location = "docker.mirrors.sjtug.sjtu.edu.cn" 查看是否成功 podman info | grep Location 输出：Location: docker.mirrors.sjtug.sjtu.edu.cn
