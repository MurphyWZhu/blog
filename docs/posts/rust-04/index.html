Rust枚举和模式匹配

May 22, 2021

枚举#

定义枚举

enum IpAddrKind {
    V4,
    V6,
}

枚举值

let four = IpAddrKind::V4;
let six = IpAddrKind::V6;

他们的类型都是IpAddrKind

如函数：

fn rout(ip_type: IpAddrKind) { }

可以向下面一样调用

route(IpAddrKind::V4);
route(IpAddrKind::V6);

match控制流运算符#

enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter,
}

let a = Coin::Penny;
let b = match a {
    Coin::Penny => 1,
    Coin::Nickel => 5,
    Coin::Dime => 10,
    Coin::Quarter => 25,
};
println!("{}", b);
let c = match a {
    Coin::Penny => {
        println!("Lucky penny!");
        1
    },
    Coin::Nickel => 5,
    Coin::Dime => 10,
    Coin::Quarter => 25,
}

绑定值的模式

可以用来获取枚举绑定的值

enum Ip {
    V4(u8, u8, u8, u8),
    V6(String),
}

fn print_ip(ip: &Ip) {
    match ip {
        Ip::V4(a, b, c, d) => println!("{}.{}.{}.{}", a, b, c, d),
        Ip::V6(a) => println!("{}", a),
    }
}

fn main() {
    let ip4 = Ip::V4(127, 0, 0, 1);
    let ip6 = Ip::V6(String::from("::1"));
    print_ip(&ip4);
    print_ip(&ip6);
}

通配符

rust的match必须列出所有可能，有时候只需要几个匹配，其他的可以用通配符_来匹配

let some_u8_value = 0u8;
match some_u8_value {
    1 => println!("one"),
    3 => println!("three"),
    5 => println!("five"),
    7 => println!("seven"),
    _ => (),
}
